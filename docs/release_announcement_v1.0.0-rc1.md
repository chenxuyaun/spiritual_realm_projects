## 🚀 核心功能

### 1. 智能工作流编排
自动分析用户意图，选择最优的处理工作流，支持 5 种核心工作流：搜索问答、教学生成、多轮对话、知识库问答、自问自答搜索。

### 2. 完整的意识模块系统
实现了包括自我模型、世界模型、元认知、动机系统、情感系统等在内的完整认知架构。

### 3. 真实模型集成
支持 HuggingFace 模型，包括 Qwen-Chat、GPT-2、T5、BART 等，支持 8-bit/4-bit 量化和 FlashAttention 加速。

### 4. 高级优化和监控
集成 vLLM、DeepSpeed、ONNX、OpenVINO 等推理引擎，支持动态批处理、KV 缓存、多 GPU 并行，提供 Prometheus 监控和 OpenTelemetry 追踪。

### 5. 完整的 API 和 CLI
提供 RESTful API 和功能完整的命令行接口，支持认证授权和统一的错误处理。

## 🎯 快速开始

### 安装

```bash
git clone <repository-url>
cd muai-orchestration
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 使用

```bash
# 单次查询
python -m mm_orch.main "什么是机器学习？"

# 交互式对话
python -m mm_orch.main --mode chat

# 启动 API 服务
python -m mm_orch.main --serve
```
